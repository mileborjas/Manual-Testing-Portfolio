# Gestión de Pruebas con Qase TMS 📊

Este apartado documenta el uso de herramientas profesionales de **Test Management System (TMS)** para la organización, diseño y ejecución de estrategias de prueba.

## 📋 Proyecto: Repositorio de Pruebas para Instagram
Se utilizó la plataforma **Qase.io** para estructurar el ciclo de vida de pruebas de funcionalidades críticas.

### Contenido de la Evidencia:
* **[Ver Reporte Detallado (PDF)](./evidencia_qase.pdf)**: Documento que incluye la exportación de los casos de prueba y los resultados de ejecución.

### Habilidades Aplicadas:
* **Diseño de Casos de Prueba:** Creación de pasos detallados, precondiciones y resultados esperados.
* **Organización de Suites:** Estructuración de pruebas por módulos (Login, Perfil, Configuración).
* **Gestión de Ejecuciones (Test Runs):** Seguimiento de estados (Pass/Fail) y trazabilidad de defectos.
* **Manejo de Herramientas TMS:** Administración integral de repositorios de pruebas en la nube.

---
*Herramientas: Qase.io, Gestión de Casos de Prueba.*
