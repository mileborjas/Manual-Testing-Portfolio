# E-commerce Manual Testing Project

## Description
Este proyecto documenta el análisis funcional y las pruebas manuales realizadas sobre una plataforma de E-commerce, cubriendo flujos críticos de negocio.

## Documentación del Proyecto
Puedes ver el reporte completo aquí: [Descargar PDF del Proyecto](./Manual_Testing_Ecommerce_Project.pdf)

## Alcance de las Pruebas
- **Flujos probados:** Login, Carrito de compras y Proceso de Checkout.
- **Artefactos incluidos:** Casos de prueba, Reporte de bugs y validaciones de API.
